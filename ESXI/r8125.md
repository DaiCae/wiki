# ESXI 网卡驱动安装(r8125) 

1. 下载驱动 https://github.com/realganfan/r8125-esxi/releases/

2. 将 VIB 文件传至 esxi 的/tmp 目录

3. 使用 esxcli 命令安装驱动

```bash
esxcli software vib install -v /tmp/Realtek_bootbank_net-r8125_9.007.01-1.vib
```

4. 重启 ESXI 主机, 查看网卡是否识别成功
